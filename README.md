# Face Recognition Attendance System (FRAS)

A desktop attendance system that uses face recognition to log employee entries and exits, with guest enrollment and security review tools.

## Download

Download the latest release from the Releases page.

## Features

- Real-time face detection and tracking
- Entry/exit detection
- Guest enrollment for testing
- Attendance logs
- Security override review
- Fullscreen desktop UI
- Windows installer support

## How to Use

1. Download `FRAS_Setup.exe` from Releases.
2. Install the app.
3. Download `yolov8n-face.pt` from Releases and place it in `models/`.
4. Open the app.
5. Use the Guest section to upload your own photos and test the system.
